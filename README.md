## Getting Started

### Prerequisites
- **Haskell (Stack)**
- **Rust (Cargo)**
- **Node.js (npm)**
- **Docker & Nix (Optional for deployment)**

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/flux.git
   cd flux
   ```

2. **Run the Setup Script:**
   ```bash
   ./scripts/setup.sh
   ```

3. **Build & Run Components:**
   - **Haskell Core:**
     ```bash
     cd core
     stack build
     stack exec flux
     ```
   - **Rust Router:**
     ```bash
     cd router
     cargo build --release
     ./target/release/router
     ```
   - **Dashboard:**
     ```bash
     cd dashboard
     npm install
     npm start
     ```

4. **Run Tests:**
   ```bash
   ./scripts/test-all.sh
   ```
